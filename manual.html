Consejos generales
1
Acceso al sistema

CircoHaus es una aplicación web. No necesitás instalar nada — abrís el navegador desde cualquier dispositivo y entrás a la dirección del sistema.

Cómo acceder
1

Abrí Chrome, Safari o Edge en tu computadora, tablet o celular.

2

Escribí la dirección web del sistema en la barra del navegador.

3

La app carga automáticamente y ya podés empezar a usarla.

Guardá la dirección como favorito en tu navegador para acceder con un solo clic desde cualquier dispositivo.
Navegación

Al entrar vas a ver la pantalla de inicio con las tarjetas de cada módulo. Hacés clic en la tarjeta del módulo que necesitás. Para volver al inicio usás el botón ← Inicio en la barra superior. En la esquina superior derecha siempre está disponible la campana de notificaciones.

Tarjetas de módulos

Seis tarjetas para acceder a cada módulo. Al hacer clic entrás directo al módulo. Desde cualquier módulo podés volver a la home con el botón ← Inicio.

3
Categorías

Las categorías organizan los productos por tipo (Aromas, Vajilla, Decoración, etc.). Conviene crearlas antes de cargar productos.

Agregar una categoría
1

Clickeá la tarjeta Categorías en la home.

2

Escribí el nombre en el campo Nombre.

3

Clickeá Guardar categoría.

4

La categoría aparece en la lista y ya está disponible al cargar productos.

No se pueden crear dos categorías con el mismo nombre.

Una vez creadas, las categorías aparecen como opciones al cargar o editar un producto y podés filtrar por ellas en Stock y Productos.

Editar un producto

Clickeá el botón naranja de edición en la fila del producto. Se abre una ventana donde podés modificar descripción, precios, margen, stock mínimo y categoría. El código no se puede cambiar.

Eliminar un producto

Clickeá el botón rojo de eliminar. El sistema pide confirmación antes de borrar.

Eliminar un producto es permanente y no se puede deshacer.
6
Ingresos y actualización de costos

Cuando llega mercadería de un proveedor registrás el ingreso acá. El sistema actualiza el stock automáticamente y, si el costo cambió, te pregunta cómo ajustar el precio.

Registrar un ingreso
1

Clickeá la tarjeta Ingresos en la home.

2

Completá la fecha, el proveedor y el número de remito (opcional pero recomendado).

3

En el buscador escribí el código o nombre del producto.

4

Seleccioná el producto de la lista que aparece.

5

Ingresá la cantidad y el precio de costo de este ingreso.

6

Clickeá + Agregar. Repetí para cada producto del ingreso.

7

Cuando terminés, clickeá Confirmar ingreso.

Al confirmar, el stock de cada producto se actualiza automáticamente.
El stock no se actualiza hasta que clickeás Confirmar ingreso. No cierres la pantalla sin confirmar.

Buscar y filtrar

Podés buscar productos escribiendo el código o nombre en el buscador. También podés filtrar por categoría usando el selector al lado del buscador.

8
Notificaciones

La campana en la esquina superior derecha te avisa cuando hay productos con stock bajo o agotado. Al abrir la app el sistema revisa el stock automáticamente. Si hay alertas aparece un número rojo sobre la campana.

Tipos de alerta
TipoSignificado
NaranjaEl stock del producto llegó al mínimo o está por debajo.
RojoEl producto está completamente sin stock.
Las notificaciones se actualizan cada vez que abrís el panel. Hacé clic en la campana para ver el detalle completo.
9
Consejos generales
Orden de carga recomendado

Al empezar a usar el sistema cargá en este orden: primero Categorías, luego Proveedores, luego Productos. Así cuando cargues un producto ya tenés disponibles todas las opciones.

Confirmá siempre el ingreso

El stock no se actualiza hasta que clickeás Confirmar ingreso. No cierres la pantalla sin confirmar o perdés el ingreso.

Aprovechá el historial de costos

Cada vez que cambia el costo de un producto, el sistema guarda el cambio con fecha. Esto va a ser muy valioso cuando agreguemos el módulo de reportes.

Usá el stock mínimo

Configurar el stock mínimo para cada producto es clave. El sistema te avisa automáticamente cuándo es momento de reponer, sin que tengas que estar revisando todo el tiempo.

El número de remito

Aunque es opcional, recomendamos cargarlo siempre. Si hay alguna diferencia con el proveedor, podés cruzar los datos fácilmente.

Acceso desde cualquier dispositivo

El sistema está adaptado para funcionar bien en computadora, tablet y celular. Guardá la dirección como favorito para acceder con un solo clic.
